## OnResetTimeout

Type
: Notification

Sender
: HMI

Purpose
: Inform SDL that a timeout has been reset via user action for some transient UI element.

### Notification

#### Parameters

|Name|Type|Mandatory|Additional|
|:---|:---|:--------|:---------|
|appID|Integer|true||
|methodName|String|true||
