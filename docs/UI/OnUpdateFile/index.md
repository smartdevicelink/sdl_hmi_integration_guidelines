## OnUpdateFile

Type
: Notification

Sender
: HMI

Purpose
: For the HMI to tell Core that a file needs to be retrieved from the app.

### Notification

#### Parameters

|Name|Type|Mandatory|Additional|
|:---|:---|:--------|:---------|
|appID|Integer|true||
|fileName|String|true|maxlength: 255|
